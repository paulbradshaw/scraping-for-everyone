Sample pages for the book Scraping for Everyone

In the book Scraping for Everyone (also published as Scraping for Journalists) I use a number of examples of live webpages that you can use scraping techniques with. Over time, many of these webpages may either change their design or their structure, or disappear altogether, which means that the code explained in the book will no longer work. If that happens, you can use the samples listed below to test that code and see it working. Note that I have only used the raw HTML, so these webpages do not include any images or other resources. However, the scrapers are only concerned with that raw HTML.
